Как создавать объекты через ассеты
В папке проекта надо нажать правой кнопкой мыши, выбираешь Create → 3D Object Sistem → Object Asset. Появляется файл. Называешь его как хочешь, например "Стул".
Кликаешь на него. В инспекторе справа есть поле Prefab туда перетаскиваешь свою 3D-модель. Всё, ассет готов , далее внутри его можно настроить.
Теперь он лежит в папке. Когда тебе надо поставить такой объект в любую сцену, просто выделяешь этот ассет и через инспектор подвязываешь его к скрипту который его принемает и загружает на сцену, в комплекте идет скрипт Test который создает все принимает такие ассаты и создает обекты на сцене.
